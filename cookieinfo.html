About this sites cookies

This site uses cookies and web beacons to store and collect data. Bellow is what these are for.

  1. Google Analytics is used to provide information to the developer about how may people are visiting this site, where they are (State, Country, etc.), etc.
  2. cookie_consent is a small key that is used when a user visits this site. It's used when a user allows cookies and is used to remove the allow cookie banner. If you would like to remove this cookie click here.
  3. User data is information that the user enters into the site. If you would like to remove this data click here. Please note this this will only remove local data, and not anything placed into a outside database. If you would like help erasing that, do not hesitate to contact me.
